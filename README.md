# 🛡️ Shield VPN Client

**Shield VPN** — это современный, быстрый и легковесный кроссплатформенный десктопный клиент для управления VPN-туннелями, построенный на базе связки движков **Electron (Node.js)** и ультрабыстрого ядра **Sing-box Core**.

Интерфейс разработан в футуристичном минималистичном стиле (Dark Mode) с адаптивной боковой шторкой навигации, предоставляя полный контроль над трафиком прямо в один клик.

## ✨ Особенности и функционал

- 🚀 **Sing-box Core Integration:** Полная поддержка протоколов нового поколения: VLESS (с Reality/gRPC/XTLS), Shadowsocks, Trojan и WireGuard.
- 📂 **Умный импорт в 1 клик:** Автоматический парсинг входящих шеринговых ссылок формата `vless://`, генерация нативного JSON-конфигуратора и поддержка удаленных URL-подписок (`https://`).
- 🎨 **Кастомный GUI интерфейс:** Стильная шторка меню, динамический вывод логов ядра в реальном времени внутри изолированного терминала и мониторинг проходящего трафика.
- 🎛️ **Автономная локальная маршрутизация:** Автоматический подъем локальных SOCKS5 (порт 10808) и HTTP (порт 10809) прокси-серверов.
- 🔄 **Интегрированная система OTA:** Прямая проверка свежих релизов приложения по нажатию кнопки через вызовы GitHub API репозитория.
